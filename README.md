# chronuro
Chronuro is a futuristic voice-controlled AI assistant built with Python. It uses AI-powered speech recognition, natural language understanding, and voice synthesis to create a real-time interactive assistant experience. Chronuro is designed to save time by helping users perform tasks quickly through simple voice interaction, making technology feel more natural and efficient.

Features

Voice-controlled interaction
Speech-to-Text using Whisper
AI responses using Gemini API
Text-to-Speech voice replies
Real-time date, day, and time
Weather updates
Web and YouTube search
WhatsApp messaging support
Email messaging
Interactive AI conversations
Modular and scalable architecture

Tech Stack

Python
Whisper STT
Google Gemini API
pyttsx3 / gTTS
SpeechRecognition

How It Works

User speaks → Chronuro listens → converts speech to text → processes the request using AI → performs the task → responds with voice output.
Chronuro is built to provide a smooth and interactive AI experience where users can naturally communicate with the assistant to save time, automate simple tasks, and access information instantly.

Installation

1. git clone https://github.com/zafarimam01/chronuro
, 2. cd chronuro
, 3. pip install -r requirements.txt
, 4. Create a .env file:
, 5. GEMINI_API_KEY=your_api_key_here
, 6. Run the assistant:
, 7. python chronuro.py
